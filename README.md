
## Introdução
    Projeto prático de backend desenvolvido no bootcamp da Labenu em janeiro de 2023. 

## Tecnologias utilizadas

1. ``Node.js``
2. ``Typescript.js``
3. ``Knex``
4. ``Express``
5. ``SQLite``

## Funcionabilidades do Projeto

- [x] Cadastro de usuário:
- id;
- email;
- password;

<br>

- [x] Cadastro de produtos:
- id;
- name;
- price;
- category;


<br>

- [x] Cadastro de compras:
- userId;
- productId;
- quantity;
- totalPrice;

<br>
